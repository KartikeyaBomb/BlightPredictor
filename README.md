# Blight Predictor

**Winner of the Memphis City Hackathon 2025 in the Blight Data Analytics Category**

## Live Deployment
[https://blight-predictor.vercel.app/](https://blight-predictor.vercel.app/)

## About
This project is a comprehensive blight prediction system for Memphis, Tennessee. Using advanced data analytics and algorithms, we help city staff and community members prioritize properties for blight mitigation efforts.

## Hackathon
This project was created during the [Memphis City Hackathon 2025](https://memphistn.gov/city-of-memphis-hackathon/).

## Features
- Interactive map visualization of blight risk areas
- Data-driven predictions using multiple data sources
- User-friendly interface for city staff and community members
- Comprehensive analytics and reporting

## Tech Stack
- React
- Mapbox GL JS
- Pandas, and NumPy

## Data Sources
- 3 different data sources
- 30,604 data entries
- 4,853 properties analyzed

## Mission
Empowering Memphis with advanced data analysis to prioritize properties for blight mitigation efforts, turning disconnected events into early warnings and actionable priorities.
